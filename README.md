# Squishy Letters

A playful desktop-only experiment in real-time 3D and hand-tracking.  
Type any text and watch it inflate into interactive, squishy letters you can manipulate via webcam or with your cursor.

Live demo: [https://squishyletters.com](https://squishyletters.com)

---

## Tech

- HTML / CSS / JavaScript  
- Three.js for 3D rendering  
- MediaPipe Hands for webcam interaction  
- Cursor-based orbit, zoom, and hover interactions

---

## Notes

Desktop-only; webcam interactions don’t work on mobile.
